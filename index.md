### Installation

1. Téléchargement
```bash
git clone <url>
cd Emulator-68k/
```

2. Tests unitaires 
```bash
make unit_test
./target/build/unit_test
```

3. Build 
```bash
make exec
./target/build/exec
```

4. Build debug
```bash
make debug
./target/build/debug
```
